# Trade Blotter Frontend

Vue 3, Pinia, TypeScript, and Vite client for the Trade Blotter assessment.

From this directory:

```powershell
npm install
npm run dev
```

The API base URL comes from `VITE_API_BASE_URL` in `.env` and defaults to
`http://localhost:5080`.

Useful commands:

```powershell
npm run type-check
npm test
npm run build
```

See the repository root [`README.md`](../README.md) for full-stack setup,
SQL Server and in-memory modes, API details, and submission notes.
